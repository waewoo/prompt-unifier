# Frontend CSS

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.

## When to use this skill

- When writing or modifying CSS, SCSS, Sass, Less, or Stylus files
- When creating stylesheets for components, pages, or layouts
- When implementing design systems, theme variables, or CSS custom properties
- When optimizing CSS performance (reducing file size, eliminating unused styles)
- When using CSS Grid or Flexbox for layout positioning
- When working with CSS variables (`--custom-property`) and custom properties
- When writing CSS selectors and rules with proper specificity
- When organizing CSS modules, partials, or component-scoped styles
- When implementing responsive styles with media queries
- When managing CSS specificity and avoiding `!important` overuse
- When using CSS preprocessors like Sass, PostCSS, or CSS-in-JS libraries
- When creating animations, transitions, or transforms
- When working with CSS frameworks or utility classes (Tailwind, Bootstrap)
