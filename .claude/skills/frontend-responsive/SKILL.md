# Frontend Responsive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.

## When to use this skill

- When building or modifying UI layouts and page structures
- When implementing CSS media queries for different screen sizes and devices
- When creating flexible grid systems with CSS Grid or Flexbox
- When optimizing layouts for mobile, tablet, and desktop viewports
- When working with responsive images using `srcset`, `sizes`, or `picture` elements
- When implementing responsive video embeds or media players
- When testing cross-device compatibility and responsive behavior
- When writing CSS with breakpoints for various screen sizes
- When using CSS Grid or Flexbox for fluid, responsive layouts
- When implementing viewport meta tags and mobile optimization
- When handling responsive typography with relative units (rem, em, %)
- When creating adaptive navigation patterns (hamburger menus, collapsible navs)
- When building mobile-first designs that scale up to larger screens
